CAMERA INDICATORS
-----------------

**Visual Layout:**
```
Cameras: ⚫ ⚫ ⚫   (0 cameras)
Cameras: 🟢 ⚫ ⚫   (1 camera)
Cameras: 🟢 🟢 ⚫   (2 cameras)
Cameras: 🟢 🟢 🟢   (3 cameras)
```

**How It Works:**
1. System checks each configured camera
2. Counts how many are available
3. Lights up that many indicators
4. First box = camera 1
5. Second box = camera 2
6. Third box = camera 3

**Benefits:**
- Visual at a glance
- No need to read numbers
- Clear indication of camera count
- Supports up to 3 cameras

ROBOT ARM INDICATORS
--------------------

**Visual Layout:**
```
Robot: ⚫ ⚫     (Not connected)
Robot: 🟢 🟢    (Connected)
```

**How It Works:**
1. Checks if USB port exists
2. If yes: Both boxes GREEN
3. If no: Both boxes RED/GRAY

**Benefits:**
- Clear connected/disconnected state
- Two boxes = more visible
- Consistent with camera layout
- Easy to spot issues

INDICATOR LOGIC
---------------

### Camera Indicators:
```python
# Light up based on count
indicator1.green if count >= 1
indicator2.green if count >= 2
indicator3.green if count >= 3

# Examples:
0 cameras: ⚫⚫⚫
1 camera:  🟢⚫⚫
2 cameras: 🟢🟢⚫
3 cameras: 🟢🟢🟢
```

### Robot Indicators:
```python
# Both on or both off
if robot_connected:
    indicator1.green
    indicator2.green
else:
    indicator1.red
    indicator2.red
```
