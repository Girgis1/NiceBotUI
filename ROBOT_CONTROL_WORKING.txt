COMMANDS AVAILABLE
------------------
Test connection:

Read current position:
  python3 rest_pos.py --read

Move to home (2048) and disable torque:
  python3 rest_pos.py --go

Save current position as new home:
  python3 rest_pos.py --save

GUI INTEGRATION
---------------
The GO HOME button in the GUI now calls rest_pos.py --go
This will:
1. Move all motors to position 2048 (middle/neutral)
2. Wait for movement to complete
3. Disable torque so motors are relaxed
4. You can manually position the robot after this

TO SAVE A NEW HOME POSITION
----------------------------
1. Manually move robot to desired position
2. Run: python3 rest_pos.py --save
3. The current positions will be saved to config.json
4. Next time GO HOME is pressed, it goes to your saved position

CONFIG
------
Home position is stored in config.json:
  "rest_position": {
    "positions": [2048, 2048, 2048, 2048, 2048, 2048],
    "speed_scale": 0.6,
    "disable_torque_on_arrival": true
  }

You can edit these values manually or use --save to capture current position.
