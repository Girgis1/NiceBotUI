GUI INTEGRATION
---------------
The GO HOME button in the GUI now calls rest_pos.py --go
This will:
1. Move all motors to position 2048 (middle/neutral)
2. Wait for movement to complete

4. You can manually position the robot after this

TO SAVE A NEW HOME POSITION
----------------------------
1. Manually move robot to desired position
2. Run: python3 rest_pos.py --save
3. The current positions will be saved to config.json
4. Next time GO HOME is pressed, it goes to your saved position
