HOW TO USE
----------

### Add Cameras:
1. Open Settings (⚙️)
2. Go to "Cameras" tab
3. You'll see your existing cameras loaded
4. Click "➕ Add Camera"
5. Fill in name, device, resolution, fps
6. Click "➕ Add Camera" again for more
7. Remove any camera with 🗑️ button
8. Click Save

### Add Teleop Arms:
1. Open Settings (⚙️)
2. Go to "Teleop Arms" tab
3. You'll see your existing teleop arm(s)
4. Click "➕ Add Teleop Arm" 
5. Fill in name, type, port, ID
6. Click "➕ Add Teleop Arm" again for more
7. Remove any arm with 🗑️ button
8. Click Save

### Config File Format:
```json
{
  "cameras": {
    "front": {
      "type": "opencv",
      "index_or_path": "/dev/video0",
      "width": 640,
      "height": 480,
      "fps": 30
    },
    "wrist": {
      "type": "opencv",
      "index_or_path": "/dev/video2",
      "width": 640,
      "height": 480,
      "fps": 30
    },
    "top": {
      "type": "opencv",
      "index_or_path": 0,
      "width": 1280,
      "height": 720,
      "fps": 30
    }
  },
  "teleop": {
    "leader1": {
      "type": "so100_leader",
      "port": "/dev/ttyACM1",
      "id": "leader_arm_1"
    },
    "leader2": {
      "type": "so100_leader",
      "port": "/dev/ttyUSB0",
      "id": "leader_arm_2"
    }
  }
}
```
