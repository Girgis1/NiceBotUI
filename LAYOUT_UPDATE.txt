TECHNICAL DETAILS
-----------------

**resizeEvent Implementation:**
```python
def resizeEvent(self, event):
    super().resizeEvent(event)
    # Make home button square (width = height)
    if hasattr(self, 'home_btn'):
        height = self.home_btn.height()
        if height > 0:
            self.home_btn.setFixedWidth(height)
```

**Benefits:**
- Triggers on any window resize
- Safe check with hasattr
- Only sets width when height is valid
- No infinite loop (setFixedWidth doesn't trigger resize)

**Settings Button Layout:**

history_and_settings = QHBoxLayout()
